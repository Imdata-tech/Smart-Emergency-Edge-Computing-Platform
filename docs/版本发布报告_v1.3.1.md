## 🎯 版本概述

CardioGuard 360 v1.3.1 是**ECG心电图前端可视化的重要版本**,实现了:

1. ✅ ECG 实时波形 ECharts 显示

## ✨ 核心功能

### 1. ECG 监测页面

#### 页面布局
```
┌─────────────────────────────────────────────┐
│  统计卡片区域 (4个卡片)                       │
│  [当前心率] [采样率] [记录时长] [连接状态]     │
├─────────────────────────────────────────────┤
│  ECG 波形图 (ECharts Line Chart)             │
│  [开始记录] [停止记录] [AI 分析] 按钮         │
├──────────────────┬──────────────────────────┤
│  心电参数详情     │  AI 分析结果展示          │
│  - PR间期        │  - 诊断结果               │
│  - QRS时限       │  - 心律失常类型           │
│  - QT间期        │  - AI置信度               │
│  - 电轴          │  - 风险等级               │
│  - HRV           │  - 建议措施               │
│  - 数据质量      │  - 医生审核状态           │
├─────────────────────────────────────────────┤
│  分析历史记录表格 (分页)                      │
│  [ID] [诊断] [类型] [置信度] [风险] [操作]    │
└─────────────────────────────────────────────┘
```

#### 统计卡片
- **当前心率** - 实时显示心率值 (bpm)
- **采样率** - ECG 采样频率 (Hz)
- **记录时长** - 当前记录持续时间 (秒)
- **连接状态** - WebSocket 连接状态指示

### 2. ECG 波形可视化

#### ECharts 配置
```typescript
{
  tooltip: { trigger: 'axis' },
  grid: { left: '3%', right: '4%', bottom: '3%', top: '10%' },
  xAxis: { type: 'category', boundaryGap: false },
  yAxis: { type: 'value', name: '电压 (mV)', min: -2, max: 2 },
  dataZoom: [
    { type: 'inside', start: 0, end: 100 },
    { start: 0, end: 100 } // 手柄拖动
  ],
  series: [{
    type: 'line',
    smooth: false,
    showSymbol: false,
    lineStyle: { color: '#409eff', width: 2 },
    areaStyle: {
      color: {
        type: 'linear',
        colorStops: [
          { offset: 0, color: 'rgba(64, 158, 255, 0.3)' },
          { offset: 1, color: 'rgba(64, 158, 255, 0.05)' }
        ]
      }
    }
  }]
}
```

#### 特性
- ✅ 实时数据更新 (模拟生成)
- ✅ 保持最近 500 个数据点
- ✅ 支持鼠标滚轮缩放
- ✅ 支持手柄拖动查看
- ✅ 渐变色填充效果
- ✅ 响应式自适应 (autoresize)

### 3. 心电参数展示

| 参数 | 说明 | 正常范围 |
|------|------|---------|
| PR间期 | 心房除极到心室除极时间 | 120-200 ms |
| QRS时限 | 心室除极时间 | 60-100 ms |
| QT间期 | 心室除极+复极总时间 | 350-450 ms |
| 电轴 | 心室除极平均方向 | -30° to +90° |
| HRV | 心率变异性 | 20-50 ms |
| 数据质量 | 信号质量评分 | 0-100 |

### 4. AI 分析结果

#### 诊断结果
- **NORMAL** (正常) - 绿色标签
- **ABNORMAL** (异常) - 红色标签

#### 心律失常类型
- **AFIB** - 房颤
- **PVC** - 室性早搏
- **PAC** - 房性早搏
- **ST_ELEVATION** - ST段抬高

#### 风险等级
- **LOW** (低风险) - 绿色
- **MEDIUM** (中风险) - 黄色
- **HIGH** (高风险) - 红色
- **CRITICAL** (危急) - 深红色

#### 医生审核状态
- **PENDING** (待审核) - 黄色
- **APPROVED** (已通过) - 绿色
- **REJECTED** (已拒绝) - 红色

### 5. 分析历史记录

#### 表格列
- ID - 分析结果ID
- 诊断结果 - NORMAL/ABNORMAL
- 心律失常 - AFIB/PVC/PAC/ST_ELEVATION
- 置信度 - AI 置信度百分比
- 风险等级 - LOW/MEDIUM/HIGH/CRITICAL
- 审核状态 - PENDING/APPROVED/REJECTED
- 分析时间 - 创建时间
- 操作 - 查看报告/下载报告

#### 分页功能
- 每页显示 10 条记录
- 支持翻页查询
- 自动刷新数据

### 6. 诊断报告功能

#### 查看报告
- 点击"查看报告"按钮
- 在新窗口显示 Markdown 格式报告
- 包含完整诊断信息和医生审核意见

#### 下载报告
- 点击"下载"按钮
- 自动下载 .md 文件
- 文件名: `ECG_Report_{resultId}.md`

## ⚠️ 已知限制

### 当前版本限制

1. **ECG数据为模拟生成**
   - 目前使用正弦波模拟
   - 未接入真实设备数据
   - 需要WebSocket实时推送(v1.3.2)

2. **心电参数为模拟值**
   - PR/QRS/QT间期为随机生成
   - 需要从后端API获取真实计算结果

3. **缺少多导联支持**
   - 目前仅显示单导联
   - 需要扩展为12导联同步显示

### 后续改进方向

1. **WebSocket实时数据**
   - 接入真实ECG设备
   - 毫秒级数据推送
   - 多用户并发支持

2. **高级波形分析**
   - R波自动检测
   - 心率自动计算
   - 异常波形标注

3. **多导联同步**
   - 12导联ECG显示
   - 同步滚动和缩放
   - 导联切换功能
