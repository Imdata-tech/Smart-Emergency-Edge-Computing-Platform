## 🚀 部署准备

### 开发环境
```bash
# 安装依赖
cd cardio-guard-frontend
npm install

# 启动开发服务器
npm run dev

# 运行测试
npm test
npm run test:coverage
```

### 生产构建
```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

### Docker 部署
```bash
# 构建镜像
docker build -t cardioguard-frontend:1.2.0 .

# 运行容器
docker run -p 80:80 cardioguard-frontend:1.2.0
```
