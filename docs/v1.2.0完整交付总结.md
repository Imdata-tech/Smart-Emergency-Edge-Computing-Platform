## 🚀 部署与运行

### 开发环境启动

```bash
# 1. 安装依赖
cd cardio-guard-frontend
npm install

# 2. 启动开发服务器
npm run dev
# 访问 http://localhost:3000

# 3. 运行测试
npm test
npm run test:coverage

# 4. 代码检查
npm run lint
```

### 生产构建

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

### Docker 部署

```bash
# 构建镜像
docker build -t cardioguard-frontend:1.2.0 .

# 运行容器
docker run -p 80:80 cardioguard-frontend:1.2.0
```

### Nginx 配置要点

```nginx
server {
    listen 80;
    server_name cardioguard.example.com;
    root /usr/share/nginx/html;
    
    # SPA 路由支持
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # API 代理
    location /api {
        proxy_pass http://backend:8080;
    }
    
    # WebSocket 代理
    location /ws {
        proxy_pass http://backend:8080;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```
