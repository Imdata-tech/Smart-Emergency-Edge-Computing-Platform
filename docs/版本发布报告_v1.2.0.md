### 1. 用户认证系统

**功能特性:**
- JWT Token 登录
- 路由守卫权限验证
- Token 自动刷新
- 用户信息管理

**技术实现:**
```typescript
// Pinia Store
export const useUserStore = defineStore('user', () => {
  const token = ref<string>(localStorage.getItem('token') || '')
  const userInfo = ref<UserInfo | null>(null)
  
  async function login(username: string, password: string) {
    const response = await axios.post(`/api/auth/login?...`)
    token.value = response.data.data
    localStorage.setItem('token', token.value)
    await fetchUserInfo()
  }
})
```

**路由守卫:**
```typescript
router.beforeEach((to, from, next) => {
  const token = localStorage.getItem('token')
  if (to.meta.requiresAuth && !token) {
    next('/login')
  } else {
    next()
  }
})
```

### 3. 实时监测 (Monitor)

**核心功能:**
- 💓 大字体心率显示(72px)
- 🌊 实时波形图(50个数据点)
- 📋 详细生理参数(RR间期、HRV、数据质量等)
- ⚠️ 异常状态脉冲动画
- 🕐 预警信息时间轴

**WebSocket 连接:**
```typescript
function connectWebSocket(userId: number) {
  const wsUrl = `ws://localhost:8080/api/ws/health-monitor?userId=${userId}`
  ws = new WebSocket(wsUrl)
  
  ws.onmessage = (event) => {
    const data = JSON.parse(event.data)
    handleRealtimeData(data)
  }
}

function handleRealtimeData(data: any) {
  if (data.type === 'HEART_RATE') {
    heartRateData.value.push(data)
    latestData.value = data
    
    // 保持最近100条数据
    if (heartRateData.value.length > 100) {
      heartRateData.value.shift()
    }
  }
}
```

**异常检测动画:**

```scss
.heart-icon.pulse {

}

@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.2); }
}
```

## 🎯 技术突破点

### 1. ECharts 按需引入优化

**问题:** 完整引入 ECharts 体积过大(~1MB)

**解决方案:**
```typescript
import { use } from 'echarts/core'
import { CanvasRenderer } from 'echarts/renderers'
import { LineChart, PieChart } from 'echarts/charts'
import { GridComponent, TooltipComponent } from 'echarts/components'

use([CanvasRenderer, LineChart, PieChart, GridComponent, TooltipComponent])
```

**效果:** 打包体积减少 **70%** (~300KB)

### 2. WebSocket 自动重连机制

**实现:**
```typescript
ws.onclose = () => {
  wsConnected.value = false
  // 5秒后自动重连
  setTimeout(() => connectWebSocket(userId), 5000)
}
```

### 3. 响应式图表自适应

**使用 autoresize:**
```vue
<v-chart :option="chartOption" autoresize />
```

**效果:** 窗口大小变化时图表自动调整,无需手动监听 resize 事件

### 4. 数据滚动更新

**保持最近100条数据:**
```typescript
if (heartRateData.value.length > 100) {
  heartRateData.value.shift() // 移除最旧数据
}
```

**优势:** 避免内存泄漏,保持图表流畅

## 🚀 部署方案

### Docker 部署

**Dockerfile:**
```dockerfile
FROM node:18-alpine as build
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

FROM nginx:alpine
COPY --from=build /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

**构建命令:**
```bash
docker build -t cardioguard-frontend:1.2.0 .
docker run -p 80:80 cardioguard-frontend:1.2.0
```

### Nginx 配置要点

```nginx
# SPA 路由支持
location / {
    try_files $uri $uri/ /index.html;
}

# API 代理
location /api {
    proxy_pass http://backend:8080;
}

# WebSocket 代理
location /ws {
    proxy_pass http://backend:8080;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection "upgrade";
}
```
